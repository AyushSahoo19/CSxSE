---
term: "CSS Flexbox / Grid"
category: "Frontend Engineering"
level: "Foundational"
---

Modern CSS layout systems. Flexbox: 1D layouts (rows OR columns). Grid: 2D layouts (rows AND columns simultaneously).

### Example

Flexbox: display:flex; justify-content:space-between; — items spread evenly in a row. Grid: display:grid; grid-template-columns:1fr 2fr 1fr; — 3 columns with different widths.
