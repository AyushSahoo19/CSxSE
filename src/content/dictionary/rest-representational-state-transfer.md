---
term: "REST (Representational State Transfer)"
category: "Networking & Web"
level: "Intermediate"
---

An architectural style for APIs using HTTP methods on resources identified by URLs. Stateless, cacheable, uniform interface.

### Example

GET /api/users/42 — retrieve user 42. POST /api/users — create a user. PUT /api/users/42 — update user 42. DELETE /api/users/42 — delete user 42.
