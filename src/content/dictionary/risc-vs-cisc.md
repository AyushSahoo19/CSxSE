---
term: "RISC vs CISC"
category: "Computer Architecture & Hardware"
level: "Advanced"
---

Two different processor architecture philosophies. RISC (Reduced Instruction Set Computer) uses simple, fast instructions. CISC (Complex Instruction Set Computer) uses complex instructions that can execute multiple operations in one step.

### Example

ARM processors (like Apple Silicon) use RISC, focusing on energy efficiency. Intel x86 processors use CISC, packing complex functions into single CPU instructions.
