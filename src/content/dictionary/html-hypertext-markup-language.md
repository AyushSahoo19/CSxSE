---
term: "HTML (HyperText Markup Language)"
category: "Frontend Engineering"
level: "Foundational"
---

The standard markup language for creating web page structure. Defines elements like headings, paragraphs, links, images, forms.

### Example

<h1>Welcome</h1><p>This is a paragraph.</p><a href='/about'>About Us</a> — HTML provides the skeleton; CSS provides the skin; JS provides the muscles.
