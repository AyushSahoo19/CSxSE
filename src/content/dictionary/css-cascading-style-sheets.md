---
term: "CSS (Cascading Style Sheets)"
category: "Frontend Engineering"
level: "Foundational"
---

The language for styling web pages: layout, colors, fonts, spacing, animations. 'Cascading' means styles can inherit and override.

### Example

body { font-family: 'Helvetica'; color: #333; } .button { background: blue; border-radius: 8px; } — transforms raw HTML into a beautiful design.
