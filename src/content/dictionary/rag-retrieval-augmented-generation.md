---
term: "RAG (Retrieval-Augmented Generation)"
category: "AI / ML Foundations"
level: "Advanced"
---

Combining a retrieval system (search engine) with a generative model (LLM). The model retrieves relevant documents and uses them to generate informed responses.

### Example

User asks about company policy. RAG system: (1) searches company knowledge base, (2) retrieves relevant docs, (3) LLM generates answer using those docs as context.
