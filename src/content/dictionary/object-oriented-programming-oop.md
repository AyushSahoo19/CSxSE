---
term: "Object-Oriented Programming (OOP)"
category: "OOP & Design Patterns"
level: "Foundational"
---

A programming paradigm that organizes code around 'objects' — bundles of data (attributes) and behavior (methods) that model real-world entities.

### Example

A Car object has attributes (color, speed, fuel) and methods (accelerate, brake, refuel). You think in terms of things, not procedures.
