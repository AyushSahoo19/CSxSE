---
term: "Port"
category: "Networking & Web"
level: "Foundational"
---

A logical endpoint for network communication. Each port number (0-65535) identifies a specific service running on a machine.

### Example

Port 80 = HTTP, 443 = HTTPS, 22 = SSH, 5432 = PostgreSQL. Like apartment numbers in a building — the IP is the building address, ports are units.
