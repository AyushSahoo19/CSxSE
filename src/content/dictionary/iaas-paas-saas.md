---
term: "IaaS / PaaS / SaaS"
category: "Cloud & Scalability"
level: "Foundational"
---

IaaS: rent infrastructure (VMs, storage). PaaS: rent platform (deploy code, provider manages OS/runtime). SaaS: use finished software (Gmail, Slack).

### Example

IaaS: AWS EC2 (you manage OS, runtime, app). PaaS: Heroku (you deploy code, it manages everything else). SaaS: Google Docs (you just use it).
