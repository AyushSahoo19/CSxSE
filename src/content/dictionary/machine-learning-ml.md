---
term: "Machine Learning (ML)"
category: "AI / ML Foundations"
level: "Foundational"
---

A subset of AI where systems learn from data to improve performance on a task without being explicitly programmed. Learns patterns, makes predictions.

### Example

Instead of writing rules for spam detection (if contains 'viagra'...), feed the ML model 10,000 spam and 10,000 non-spam emails. It learns the patterns itself.
