---
term: "Disjoint Set / Union-Find"
category: "Data Structures"
level: "Advanced"
---

A data structure that tracks elements partitioned into non-overlapping sets. Supports fast union (merge two sets) and find (which set contains element?).

### Example

Social network groups — union(Alice, Bob) puts them in the same group. find(Alice) == find(Bob)? tells if they're connected.
