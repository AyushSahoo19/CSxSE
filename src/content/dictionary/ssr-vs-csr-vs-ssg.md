---
term: "SSR vs CSR vs SSG"
category: "Frontend Engineering"
level: "Intermediate"
---

SSR (Server-Side Rendering): HTML generated on each request. CSR (Client-Side Rendering): JS builds HTML in browser. SSG (Static Site Generation): HTML generated at build time.

### Example

SSR: Next.js renders pages on the server — fast first load, good SEO. CSR: React SPA — blank page until JS loads. SSG: Gatsby — pre-built HTML, fastest but static.
