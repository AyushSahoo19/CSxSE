---
term: "Responsive Design"
category: "Frontend Engineering"
level: "Foundational"
---

Designing web pages that adapt to different screen sizes (mobile, tablet, desktop) using flexible layouts, media queries, and relative units.

### Example

A 3-column layout on desktop becomes 1 column on mobile. @media (max-width: 768px) { .grid { grid-template-columns: 1fr; } } — CSS adapts the layout.
