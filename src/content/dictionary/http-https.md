---
term: "HTTP / HTTPS"
category: "Networking & Web"
level: "Foundational"
---

HTTP is the protocol for web communication. HTTPS is HTTP with TLS encryption for security. Methods: GET (read), POST (create), PUT (update), DELETE (remove).

### Example

You type a URL → your browser sends an HTTP GET request → the server responds with HTML. HTTPS encrypts the conversation so hackers can't eavesdrop.
