---
term: "Strongly vs Weakly Typed"
category: "Fundamentals"
level: "Intermediate"
---

Strongly typed languages don't allow implicit type conversions (Python: '5' + 3 = error). Weakly typed languages do (JavaScript: '5' + 3 = '53').

### Example

JavaScript's '5' + 3 = '53' has caused countless production bugs. Python forces you to be explicit: int('5') + 3 = 8.
