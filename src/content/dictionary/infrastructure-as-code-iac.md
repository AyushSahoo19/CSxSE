---
term: "Infrastructure as Code (IaC)"
category: "DevOps & Infrastructure"
level: "Intermediate"
---

Managing and provisioning infrastructure through code (not manual UI clicks). Version-controlled, reproducible, and auditable.

### Example

Terraform file: 'Create 3 EC2 instances, a load balancer, and an RDS database.' Run it, infrastructure appears. Destroy it, start fresh. Like a recipe for servers.
