---
term: "SPA (Single Page Application)"
category: "Frontend Engineering"
level: "Intermediate"
---

A web app that loads a single HTML page and dynamically updates content without full page reloads. Navigation is handled by JavaScript.

### Example

Gmail: click on an email, the content loads without the page reloading. The URL changes, but the shell (sidebar, header) stays in place. React Router enables this.
