---
term: "NP-Completeness"
category: "Algorithms"
level: "Advanced"
---

A class of problems where no known polynomial-time algorithm exists, but a solution can be VERIFIED in polynomial time. If you solve one, you solve them all.

### Example

Traveling Salesman: given 50 cities, find the shortest route visiting all. Checking a given route is easy, but finding the best is astronomically hard.
