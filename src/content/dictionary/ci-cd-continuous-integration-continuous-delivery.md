---
term: "CI/CD (Continuous Integration / Continuous Delivery)"
category: "DevOps & Infrastructure"
level: "Intermediate"
---

CI: automatically building and testing code whenever changes are pushed. CD: automatically deploying tested code to production (or staging).

### Example

Push to GitHub → GitHub Actions runs tests → if green, auto-deploys to staging → after approval, deploys to production. Code reaches users in minutes, not months.
