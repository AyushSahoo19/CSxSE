---
term: "End-to-End (E2E) Testing"
category: "Testing & QA"
level: "Intermediate"
---

Testing the complete application from the user's perspective. A robot simulates real user actions in a real browser.

### Example

Cypress/Playwright: open browser → navigate to login → type credentials → click login → verify dashboard loads → click 'create post' → verify post appears. Full user flow.
