---
term: "Supervised Learning"
category: "AI / ML Foundations"
level: "Foundational"
---

Learning from labeled data — each training example has an input AND the correct output (label). The model learns to map inputs to outputs.

### Example

Training data: (photo of cat, 'cat'), (photo of dog, 'dog'). Model learns features that distinguish cats from dogs. Given a new photo, it predicts the label.
