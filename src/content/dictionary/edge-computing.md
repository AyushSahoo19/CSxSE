---
term: "Edge Computing"
category: "Cloud & Scalability"
level: "Advanced"
---

Processing data near the source (edge of the network) instead of sending everything to a central cloud. Reduces latency and bandwidth.

### Example

Self-driving cars process sensor data locally (can't wait for cloud round-trip). CDN edge functions (Cloudflare Workers) run code near users for <10ms latency.
