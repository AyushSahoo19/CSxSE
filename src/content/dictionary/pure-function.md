---
term: "Pure Function"
category: "Programming Languages & Paradigms"
level: "Intermediate"
---

A function that: (1) always returns the same output for the same input, and (2) has no side effects (doesn't modify external state).

### Example

add(3, 5) always returns 8 — pure. getCurrentTime() returns different values — impure. Math.random() — impure. Pure functions are easy to test and reason about.
