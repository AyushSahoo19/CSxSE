---
term: "Object Storage (S3)"
category: "Cloud & Scalability"
level: "Intermediate"
---

Storing data as objects (file + metadata) in a flat namespace. Highly durable, scalable, and cheap. Not a file system — no directories, just keys.

### Example

Store user profile photos in S3: key = 'users/42/avatar.jpg'. 99.999999999% (11 nines) durability. Virtually unlimited storage. Pennies per GB/month.
