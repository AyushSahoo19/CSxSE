---
term: "DevOps"
category: "DevOps & Infrastructure"
level: "Intermediate"
---

A culture and set of practices that unifies software development (Dev) and IT operations (Ops). Aims for shorter release cycles, higher quality, and faster feedback.

### Example

Instead of developers throwing code 'over the wall' to an ops team, both collaborate: automated testing, CI/CD, monitoring, infrastructure as code.
