---
term: "AWS (Amazon Web Services)"
category: "Cloud & Scalability"
level: "Intermediate"
---

The largest cloud platform with 200+ services. Key services: EC2 (compute), S3 (storage), RDS (managed database), Lambda (serverless), CloudFront (CDN).

### Example

A typical AWS stack: CloudFront (CDN) → ALB (load balancer) → ECS (containers) → RDS (database) → S3 (file storage) → SQS (message queue).
