---
term: "DNS (Domain Name System)"
category: "Networking & Web"
level: "Foundational"
---

Translates human-readable domain names (google.com) into IP addresses (142.250.80.46) that computers use. The internet's phone book.

### Example

You type 'youtube.com' → DNS lookup → 142.250.80.46 → your browser connects to that IP. Without DNS, you'd need to memorize IP addresses.
