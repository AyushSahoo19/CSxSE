---
term: "Data Warehouse vs Data Lake"
category: "Databases & Storage"
level: "Advanced"
---

Warehouse: structured, processed data optimized for analytical queries (BigQuery, Redshift). Data Lake: raw data in any format stored cheaply (S3, HDFS).

### Example

Data Lake = throwing all raw logs, CSVs, images into a giant bucket. Data Warehouse = cleaned, organized data ready for 'show me Q3 revenue by region' queries.
