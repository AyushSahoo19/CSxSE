term: "React"
category: "Frontend Engineering"

A JavaScript library for building user interfaces with reusable components. Uses JSX (HTML in JavaScript), virtual DOM, and unidirectional data flow.

### Example

function UserCard({ name, avatar }) { return <div><img src={avatar}/><h2>{name}</h2></div>; } — a reusable component. Use <UserCard name='Alice' /> anywhere.
