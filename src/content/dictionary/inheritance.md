---
term: "Inheritance"
category: "OOP & Design Patterns"
level: "Foundational"
---

A mechanism where a new class (child) inherits attributes and methods from an existing class (parent). Enables code reuse and hierarchy.

### Example

class Animal { eat() } → class Dog extends Animal { bark() }. Dog inherits eat() and adds bark(). You don't rewrite eat() for every animal.
