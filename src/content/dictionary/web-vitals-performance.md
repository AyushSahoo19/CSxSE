---
term: "Web Vitals / Performance"
category: "Frontend Engineering"
level: "Intermediate"
---

Google's metrics for web page quality. LCP (Largest Contentful Paint): loading. FID (First Input Delay): interactivity. CLS (Cumulative Layout Shift): stability.

### Example

LCP < 2.5s = good. Your hero image takes 4s to load? Optimize it: compress, use WebP format, lazy load below-fold images, use a CDN.
