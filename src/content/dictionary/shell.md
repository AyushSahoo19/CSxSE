---
term: "Shell"
category: "Operating Systems"
level: "Foundational"
---

A command-line interface for interacting with the OS. Interprets user commands and executes them. The user-facing wrapper around the kernel.

### Example

Bash, Zsh, PowerShell. You type 'ls -la' (list files), the shell interprets it, calls the kernel, and displays results. Scripts automate sequences of commands.
