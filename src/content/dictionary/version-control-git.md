---
term: "Version Control (Git)"
category: "DevOps & Infrastructure"
level: "Foundational"
---

A system that tracks changes to files over time. Git is the dominant VCS — distributed, supports branching, merging, and collaboration.

### Example

Commit = a snapshot. Branch = a parallel timeline. Merge = combining branches. You can always go back to any previous state. Like 'undo' on steroids.
