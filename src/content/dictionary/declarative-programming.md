---
term: "Declarative Programming"
category: "Programming Languages & Paradigms"
level: "Intermediate"
---

Describing WHAT you want without specifying HOW to do it. The system figures out the steps. SQL, HTML, CSS, React JSX.

### Example

SQL: SELECT name FROM users WHERE age > 18. You don't say 'scan each row, check age field, if greater...' — the database figures out the how.
