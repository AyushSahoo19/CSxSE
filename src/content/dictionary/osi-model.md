---
term: "OSI Model"
category: "Networking & Web"
level: "Intermediate"
---

7-layer conceptual model for network communication: Physical, Data Link, Network, Transport, Session, Presentation, Application. Each layer has a specific role.

### Example

Layer 7 (Application): HTTP. Layer 4 (Transport): TCP/UDP. Layer 3 (Network): IP. Layer 1 (Physical): Ethernet cable. Data flows down layers when sending, up when receiving.
